Made up of syntax that surround things called Tags


Tags

Tags are basically indicators for when the computer knowsto start or end any input. This is because the compilerneed specific patterns to interpret the information given.


Some examples include <b>, <html>, <whatever>
However, all of these need to have closing tags as well. </b>, </html>, </whatever>

Links <a href = "https://source.com"> text </a>

This allows text to become a clickable link to the webaddress specified.

Images

Follow this format if you want to add images.
<img src = "source" alt = " text">

This format shows the image tag with key words src and alt.Within the quotes are the source of the image and the substitute text if the image doesn't load.

Useful tags:

<br> = break (gives a line of whitespace) -void
<p> = paragraph(a structuring tag) </p>
<div> = 'box' container </div>
<span> = inline container element </span>

Inline vs Block

Inline elements occupy the space that are within the tags.
Block elements occupy the whole space of it's container.
